# Автотест для формы регистрации DemoQA

Этот проект содержит автоматизированный тест на библиотеке **Selene** для проверки заполнения и отправки формы [Automation Practice Form](https://demoqa.com/automation-practice-form).

## Что проверяет тест:
- Корректное заполнение всех текстовых полей.
- Выбор пола и хобби.
- Полную симуляцию выбора даты рождения через календарь.
- Загрузку файла.
- Выбор штата и города из кастомных выпадающих списков.
- Итоговую проверку данных в появившемся поп-апе.

## Особенности запуска
*   **VPN:** Сайт `demoqa.com` может быть недоступен или работать нестабильно в некоторых регионах. Для стабильного прохождения теста рекомендуется использовать VPN.
*   **Браузер:** Тест настроен для запуска в браузере Google Chrome.
